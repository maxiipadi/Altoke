# Altoke Servicio Técnico 🔧

Sitio web oficial de **Altoke Servicio Técnico**, dedicado a la reparación de celulares, computadoras y dispositivos electrónicos en Santiago del Estero, Argentina.

## 📌 Descripción

Este proyecto está desarrollado en HTML, CSS y JavaScript puro. Su objetivo es ofrecer una presencia profesional online para facilitar el contacto con clientes, mostrar horarios de atención, servicios disponibles y mantener la identidad visual de la marca.

## 🚀 Características

- Página estática rápida y liviana.
- Footer modular reutilizable con `components.js`.
- Diseño responsive para dispositivos móviles.
- Fácil de mantener y expandir.
- Ideal para talleres técnicos y servicios locales.

## 📂 Uso del componente Footer

1. Asegurate de tener este div en tus HTML:

```html
<div id="footer-component"></div>
<script src="/public/components/components.js"></script>

2. Estilos en el <head>:

<link rel="stylesheet" href="/public/components/footer.css">

3. Para modificar el contenido, editá public/components/footer.html.
